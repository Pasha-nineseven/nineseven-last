Оптимальное предложение

В качестве основы для нашей работы мы используем систему управления сайтом 1С-Битрикс. К данной системе мы разработали модули, которые решают типовые задачи автодилеров и ускоряют сборку проекта без потери в качестве.

Вы получите настроенную систему, стилизованную под ваш бренд и упростите поддержку сайтов для себя и ваших маркетологов.

Больше 8 лет мы работаем в тандеме с крупнейшей компанией по продвижению и интернет-маркетингу в Беларуси. Совместными усилиями мы делаем проекты которые легко обновлять, продвигать и развивать. Базовые требования к успешному сео-продвижению проекта вынесены в технические стандарты разработки.

Каталоги автомобилей с подборщиком

Проработанная структура хранения каталога, которая подойдет под любой дизайн.


Проект готов к продвижению

Редиректы, правила генерации мета-данных страниц и каталога, генерация карты сайта — данные подходы отработаны совместно со специалистами Webcom-Media.

Автообновление данных

Данные можно добавлять на сайт как в ручном режиме, так и автоматически через файлы формата csv, xml. Для конкретных дилеров подключаем системы: SAP, 1C 8й версии, Битрикс24.
Интеграции с SAP, 1C и PPS по протоколу SOAP

Интегарции по HTTPS используя принципы REST и формат JSON

Публичное API

Передача данных по автомобилям, ценам и комплектациям на внешние сервисы используя публичный API в формате JSON.
